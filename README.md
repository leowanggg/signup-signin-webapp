# signup-signin-webapp
## 一个基于nodejs, sailsjs和mongoDB的注册登录web应用
### 简介
- #### 框架： nodejs和sailsjs；
- #### UI： Bootstrap和jQuery；
- #### 数据库： mongoDB；
- #### 功能：
    - ##### 管理员：
        - 管理注册账户；
        - 管理权限；
    - ##### 普通游客：
        - 注册/登录；

### 使用
- `$ npm install`
- `$ sails lift`

### PS:
- 启动server前需先启动mongoDB数据库`sudo mongod --dbpath database/`
